# Zenteiq_Assignment

This repository contains my submission for the ZenteiQ ML Engineer Assignment. The work explores training transformer models using MaxText across CPU, GPU, and TPU backends, along with comparisons between Dense (Qwen) and Mixture of Experts (DeepSeek) architectures. All experiments were performed using synthetic data and include configuration changes, training logs, metrics, and analysis.

## Repository Structure

```
.
├── FINAL REPORT.pdf                 # Consolidated report with analysis and comparisons
├── README.md
├── TASK-1.zip                       # MaxText data format exploration
├── TASK-2.zip                       # Qwen3 experiments, logs, configs and analysis
├── TASK-3.zip                       # DeepSeek MoE experiments, logs, configs and analysis
├── ZENTEIQ_QWEN3_0_6B.ipynb
├── Zenteiq_qwen3_0_71B.ipynb
├── ZENTEIQ_QWEN3_1_7B.ipynb
├── ZENTEIQ_DEEPSEEK_TASK3_CPU_GPU.ipynb
└── ZENTEIQ_DEEPSEEK_TASK3_TPU.ipynb
```

## Contents

- Task 1: MaxText data format analysis
- Task 2: Dense model training using Qwen3 (0.6B and scaled 0.71B)
- Task 3: DeepSeek MoE model scaled below 1B parameters
- CPU, GPU and TPU training logs, configurations, metrics and comparative analysis
- Final consolidated report summarizing all experiments
